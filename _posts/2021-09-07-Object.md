객체(Object)를 사용할 때 알아두면 유용한 **객체 상식 가지**를 소개한다.

## 상식 1
**객체는 참조 타입이다.**

참조 타입의 변수는 실제값이 아닌 데이터의 주소값을 저장한다. 

```javascript
const car = {
  name: "Hyundai",
  model: "AVANTE",
  price: 1000,
}

console.log(car);
console.log(typeof(car));

// Expected Output:
// [object Object] {
//   model: "AVANTE",
//   name: "Hyundai",
//   price: 1000
// }
// "object"
```
